# docker-acestream-webplayer
A bland and boring static website utilising mpegts.js to connect to an instance of Ace Stream Engine and play a stream.

The artifact generated by this repo assumes the container image is mounted as an image-volume to a pod running a http webserver. For example:

```yaml
spec:
  containers:
    - name: ace-player
      image: ghcr.io/trexx/docker-busybox-httpd:1.37.1-1 
      ports:
      - name: http
        containerPort: 8080
      volumeMounts:
        - name: www
          mountPath: /www
  volumes:
    - name: www
      image:
        reference: ghcr.io/trexx/docker-acestream-webplayer:2.0.0
```
